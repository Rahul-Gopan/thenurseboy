**Thenurseboy** is a civic-driven initiative led by a passionate individual who actively cleans public places for free, conducts large-scale cleanliness drives, and unites citizens to work toward a pollution-free and beautiful nation.

This website aims to:

- Showcase achievements and impact
- Promote upcoming cleanliness events
- Encourage public participation
- Build a strong digital identity using modern web technologies
